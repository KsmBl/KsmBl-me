Projekt: custom Keyboard

Ungefähr im Februar 2023 fing ich an, mich mit dem Aufbau von Tastaturen auseinnanderzusetzen.
Auf das Thema kam ich durch einen Freund, der sich bereits seine eigene Tastatur entworfen und gelötet hat. Einen Plan für das Layout hatte ich bereits erstellt, jedoch stellte es sich als sehr schwer heraus zu diesem Plan ein PCB zu erstellenn. Daher habe ich mir ein geteilestes Ergo-Keyboard von GitHub herunter geladen und bestellt.

Auf den 2 PCBs befinden sich jeweils eine Matrixschaltung, um die vielen Tasten realisieren zu können. Daran werden dann jeweils ein Microcontroller gelötet, welcher die eingehenden Signale als Input via USB an den PC / Laptop ausgibt. Als Microcontroller habe ich zwei Elite-pi gekauft. Die beiden Einzeiltile werden dann mit einem TRRS (aka AUX) Kabel verbunden.
